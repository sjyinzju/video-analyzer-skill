# video-analyzer-skill
这是一个给obsidian里的opencode插件/claudian插件的视频分析技能，只需通过调用这个技能并输入视频网址（如YouTube、Bilibili等），opencode/Claude code会深入分析视频内容，创建视频笔记，在需要时会创建思维导图画布，旨在节省用户时间，更快更深入了解视频内容。  
This is a video analysis skill for the OpenCode plugin in Obsidian. Simply invoke this skill and enter the video URL (such as YouTube, Bilibili, etc.). OpenCode will deeply analyze the video content, create video notes, and generate a mind map canvas when needed. It aims to save users time and provide a faster and deeper understanding of the video content.

注意：这只是一个参考提示词，使用时，请把SKILL.md里面的文件路径换成你自己的。该技能同样可以给coding CLI使用，只要你安装了obsidian官方发布的skills：https://github.com/kepano/obsidian-skills  
Note: This is just a reference prompt. When using, please replace the file path in SKILL.md with your own. This skill can also be used with the coding CLI, as long as you have installed the skills officially released by Obsidian: https://github.com/kepano/obsidian-skills  

这里提供一个思路，如果有更好的视频分析的思考，也可自行添加。
